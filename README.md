## ACP-
1. **Add**: stage files for commit.

`git add -A` (*or* `*`)

2. **Commit**: snapshot of the current state of repository.

`git commit -m 'a human readable message describing changes to commit here'`

3. **Push**: Push committed files from local to remote repository. 

`git push [origin main]`


|  A  |  C  |  P  |
|-----|-----|-----|
| Add | Commit | Push |
|git add -a| git commit -m 'message here'| git push origin main|
