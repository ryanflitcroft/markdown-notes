## ACP-
1. **Add**: stage files for commit.

`git add -A` (*or* `*`)

2. **Commit**: commit file changes to push.

`git commit -m 'a human readable message describing changes to commit here'`

3. **Push**: Push committed files from local to remote repository. 

`git push [origin main]`
